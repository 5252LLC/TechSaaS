Layout Containers

Container Types

Bootstrap offers several container types to control the width of your content relative to the viewport.

.container

Default responsive fixed width container

.container-fluid

Full width container, spanning the entire viewport

.container-sm

100% width until small breakpoint

.container-md

100% width until medium breakpoint

.container-lg

100% width until large breakpoint

.container-xl

100% width until extra large breakpoint

.container-xxl

100% width until extra extra large breakpoint

Container breakpoints align with the grid system breakpoints: sm (576px), md (768px), lg (992px), xl (1200px), and xxl (1400px).
